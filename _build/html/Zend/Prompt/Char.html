687.1.1. __construct¶

__construct([$promptText = "Please hit a key"[, $allowedChars = "abc"[, $ignoreCase = true[, $allowEmpty = false[, $echo = true]]]]])¶

Ask the user for a single key stroke

Parameters:
  • $promptText (string) – The prompt text to display in console
  • $allowedChars (string) – A list of allowed chars (i.e. “abc12345”)
  • $ignoreCase (bool) – If true, case will be ignored and prompt will always return lower-cased response
  • $allowEmpty (bool) – Is empty response allowed?
  • $echo (bool) – Display the selection after user presses key
